subir los cambios.
generar los archivos con:
ng build --configuration=production --base-href=/portafolio/

 Cambiar a la rama gh-pages
 Mover los archivos con:
 Move-Item -Path dist/portafolio/* -Destination .

 ejecutar esto para agrega los archivos, haz un commit y empuja los cambios a gh-pages:
git add .
git commit -m "Actualización de la página con los últimos cambios"
git push origin gh-pages
